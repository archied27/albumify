# albumify
Organise Spotify saved albums using machine learning

## Motivation
I was motivated to make this due to Spotify's lack of features surrounding albums, and I believe listening to music via albums should be encouraged and a lot of the art of music is lost listening through playlists instead of albums

And my personal library consists of ~700 saved albums, which can be difficult to browse with Spotify's limited sorting (by artist name, album name, year, recently added), so this project will also be useful for myself
